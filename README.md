# CampusBridge

CampusBridge is a student-centric web application designed to simplify communication and academic collaboration within a college campus.

## Project Overview
CampusBridge provides a unified platform where students can connect with peers, access academic resources, and stay updated with college activities.

## Features
- One-to-one student chat  
- Group chats (Class / Branch / Year)  
- Subject-wise doubt discussion channels  
- College announcements & schedules  
- Previous year question papers  
- Notes and resource sharing  

## Technologies Used
- HTML  
- CSS  
- JavaScript  
- Figma (UI Design)

##  Application Modules
- Login Page  
- Home Dashboard  
- Student Chat & Community  
- Announcements Section  
- Academic Resources  

## How to Run the Project
1. Download or clone the repository  
2. Open the `index.html` file in any web browser  
3. The application will run locally  

##  UI Design
The user interface of the application is designed using Figma with a focus on simplicity and usability.
